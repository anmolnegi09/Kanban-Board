# Kanban Board

A responsive Kanban Board built using HTML, CSS, and JavaScript.

## Features
- 📋 Create, edit, and delete tasks
- 🖱️ Drag and drop tasks between columns
- 💾 Local Storage support
- 📊 Real-time task count updates
- 🎨 Clean and responsive UI

## Tech Stack
- HTML5
- CSS3
- JavaScript (ES6)
